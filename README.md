## ทดสอบการแก้ปัญหา
```
1. หา component react ดังต่อนี้
   - date picker ที่ไว้เลือกวัน เดือน ปี ได้ โดยถ้าเลือกเป็น พศ. จะดีมาก
   - time picker เลือกเวลาได้ 
   - table สามารถ sort row ได้ตาม โดยคลิ๊ก บน header บนตาราง และทำ grop column ได้ ปรับแต่ง style ได้
   - paging เวลาทำรายงานสามารถคลิ๊ก next last ได้
   - blocking ui ถ้ามีการ load api แล้วสามารถ block การคลิ๊ก UI
   - Autocomplete ที่สามารถ พิมพ์แล้วได้ผลลัพธ์ โดยดึงจาก api

2. นำเอา component ข้างต้น มาเขียนเป็น code ตัวอย่าง จัดวางการนำเเสนอทั้งหมดในหน้าเดียว ให้สวยงาม ดึงข้อมูล api จาก mock data ที่ตัวเองสร้างขึ้น ทดสอบการวางโครงสร้าง และการจัดวางหน้าตา ถ้า responsive ได้จะดี

3. นำ code push ขึ้น git hub พร้อมทั้ง publish example เพื่อตรวจสอบ code และหน้าตาได้

4. ก่อนทำงานให้ประเมินเวลาที่ทำ และหลังจากจบงานให้บอกระยะเวลาที่ใช้งานจริง 
```
